Inventory Management System

This mobile app was developed as a cross-platform inventory management solution using Ionic and Angular. It enables users to add, search, and view inventory items with ease.

The system integrates a RESTful API for real-time data operations and offers a professional, mobile-friendly interface.

This app was built using the Ionic Framework.

Icons are sourced from Ionicons.

Developed by Xavier Tissera and Hasindu
